# ResourceGen
ResourceGen is a tool made for SexyAppFramework (PopCap Framework), to easily edit the project's assets library.
It provides an asset tree made out of **Groups**, containing **Resources**.

## FAQ

1. Q: Does it work on every PopCap Game?

A: Yes, however if you find exceptions, please create an Issue.

2. Q: Does it work on every SexyAppFramework project?

A: Yes.

## How to use?
1. Open the Resource Manifest file in File -> Open or Create a new one in File -> New
2. Start Editing
3. Save in File -> Save
4. Export Source File (.cpp + .h) in File -> Generate Source
